======== Description ========

No more messing around with shortcodes! Easily add columns to your WordPress powered website with this 12-column responsive grid system.

== Installation ==

1. Upload the ‘righthere-visual-columns folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on ‘insert columns icon” in TinyMCE menu in WordPress
